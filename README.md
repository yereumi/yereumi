### 1. 빠르게 성장합니다
프로젝트 배포 과정에서 장애가 발생했을 때, 인프라 담당자가 부재한 상황에서도 AWS 공식 문서와 레퍼런스를 빠르게 학습하여 직접 **5시간만에** 해결한 경험이 있습니다.
또한 4주 프로젝트에서 처음 사용해본 Graph DB(Neo4j)를 단기간에 학습하고 Cypher 쿼리 기반 최적화를 적용하여 응답 속도를 **약 60% 이상** 개선했습니다.

### 2. 사용자 친화적인 서비스를 지향합니다
**약 100명의 베타 테스터**를 대상으로 성능 테스트를 진행하며, 피드백과 로그 분석을 통해 캐시 구조와 쿼리 로직을 최적화했습니다.
그 결과, 응답 속도를 **약 90% 이상** 개선하며 안정적이고 빠른 사용자 경험을 제공했습니다.

### 3. 성실합니다
학창시절 12년 개근과 대학교 우등 졸업, 그리고 LG유플러스 교육 최우수상 수상으로 성실함을 증명해왔습니다.

#### 앞으로도 성실함과 꾸준한 학습을 바탕으로 기술과 책임을 함께 고민하는 개발자로 성장하겠습니다.
